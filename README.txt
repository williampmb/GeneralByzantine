scriptbizantine.py é o script em python para executar o mininet e executar os .jar.
É preciso ajustar os caminhos para encontrar o Servidor-v2.0.jar e o ByzantineFailureModelClient-v3.3.jar dentro
do script em python.

Para executar o script é preciso passar dois parâmetros. 
Executar o script com -h ajuda a identificar quais são os parâmetros.

Os logs serão salvos na pasta onde está o script em python.

O código-fonte do General está na pasta General. O executável é o ByzantineFailureModelClient-v3.3.jar

O código-fote na pasta Servidor é um servidor BroadCast adaptado para os generais. O executável é o
Servidor-v2.0.jar

IMPORTANTE:

Adicionar ".jar" aos arquivos "ByzantineFailureModelClient-v3.3" e "Servidor-v2.0"
Adicionar ".py" aos arquivo "scriptbizantine"
